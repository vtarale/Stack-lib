# Stack-lib
stack lib for python
## usage
to import 
```python
import stack_lib.queue as stack
```
init
```python
s = stack.Stack()
```
some functions
```python
# to add
s.add(<data>)
# to remove
# Ex
item = s.get()
# to display
s.display()
# to check if is empty
s.isempty() # returns bool
```
